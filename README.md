# Tic Tac Toe React Typescript App

- React
- Redux
- React/Reducx Toolkit
- useState
- useEffect
- useSelector
- models OOP
- Jest

## Install and Run

`npm install`   - installing all modules <br />
`npm start`     - start App <br/>
default URL `http://localhost:3000/` <br />

## Test App

`npm test`      - run included tests
